R Project to process NCCA ASCII output into tidy data.

parse.r - source this script to make tidy data from the NCCA ASCII output
  this script will source the necessary scripts and functions

NCCAASCIIParse.r - contains 6 functions to parse the output and separate 
  header from data info
  
headerParse.r - contains 3 functions to parse the header into
